Rédiger un petit programme :
En HTML, créer un bouton. Utiliser l'attribut onclick pour déclencher une fonction afficherMessage.
Au click sur le bouton, afficher une alerte.
En option : encapsuler votre code dans une IIFE.

Créer plusieurs boutons en HTML. Leur mettre une classe CSS btn. En JS, parcourir tous les .btn avec une boucle. Dans le cod JS, au clique sur chaque bouton, exécuter la fonction afficherMessage ! Attention au chargement du document ! :)

Associer clicks et classes CSS

Rédiger un programme actifInactif :
Créer un petit élément d'UI (widget) représentant un utilisateur. Se servir par exemple de font-awesome et son icône user.
Au click sur l'icône, activer ou désactiver une classe CSS .is-active. Si le widget n'est pas actif, ajouter la classe, sinon la retirer.
Indice: utiliser une api déjà évoquée et testée en cours.
... element.classList !
Bonus: ajouter une transition CSS à l'action.
