Les propriétés utiles

element.children

Récupérer tous les enfants d'un élément sous forme de tableau (htmlCollection)

element.parentElement

Récupérer le parent d'un élément.

element.previousElementSibling

Récupérer le prochain frère gauche d'un élément.

element.nextElementSibling

Récupérer le prochain frère droit d'un élément.

les méthodes (functions) utiles

document.createElement(nodeName)

Créer un objet HTML.

element.appendChild(node)

Insérer un objet HTML créé avec document.createElement dans le document.

element.removeChild(enfant)

Supprime un enfant d'un élément.

element.insertBefore(nouvelElement, refElement)

Insère un élément dans l'arborescence.
