JS - let et const

En ES6 (EcmaScript 6, dernière version de JS), on dispose de deux nouvelles manière de déclarer nos variables.

Pour rappel...

La portée d'une variable, déclarée avec l'instruction var, est fonctionnelle.
Une var est donc lisible dans le scope de la fonction courante, ET ses scopes enfants.

let

Variable dont la portée est le block courant.

const

Dans le scope global, this représente window.
window est l'objet global contenant toutes les fonction natives de JS.
