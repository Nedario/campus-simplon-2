JS - La boucle forEach

Une boucle for ... avec un style "programmation fonctionnelle" !

La boucle forEach permet d'itérer sur un tableau.
Sa syntaxe peut surprendre au début, mais elle est très pratique.

Syntaxe

tableau.forEach(function clbk(el, i, arr) {

    console.log("tour n° " + (i + 1));
    console.log(el);

}, valeurDeThis);

Similaire à ...

function parcourirArr(el, i, arr) {

    console.log("tour n° " + (i + 1));
    console.log(el);
}

tableau.forEach(parcourirArr, window);

Callback

La boucle forEach prend donc une fonction en argument (un callback). Cette fonction sera exécutée pour chaque valeur du tableau, dans l'ordre de placement.

Valeur de retour

forEach retourne undefined.

Valeur de this

En plus du callback, on peut indiquer la valeur de this utilisée dans le scope du callback.
