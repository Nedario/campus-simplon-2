JSON
JavaScript Object Notation.
Inventé par Douglas Corckford (le bon Doug').
Format d'échange de plus en plus populaire, JSON c'est bon, mangez-en.
Il permet d'échanger des données structurées entre des applications, quel que soient les langages utilisés pour les coder.
Exemple: en JS, je convertis une valeur en JSON et l'envoie vers PHP. PHP pourra décoder cette valeur puis l'interpréter.
Java, Ruby, Go, Python, TypeScript, tous ces langages comprennent JSON !!!
