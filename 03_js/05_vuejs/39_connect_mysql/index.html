Connect mySQL

Dépendances du projet

  • vue js
  • mysql node
  • express js
  • nodemon

préparer le projet avec npm init.
puis en ligne de commande installer vue, express et le driver mysql pour node js.

(sudo) npm install --save vue
(sudo) npm install --save express
(sudo) npm install --save node-mysql
(sudo) npm install --save-dev nodemon

Créer un fichier server.js au même niveau que votre index.html
Dans ce fichier, placer un instruction pour afficher "hello world" dans la console. Modifier la partie scripts du fichier package.json pour qu'elle ressemble à ceci :

  "scripts": {
    "server": "nodemon server.js"
  }

depuis un terminal, dans le dossier contenant l'index de ce projet, lancer la commande suivante : 

  npm run server
