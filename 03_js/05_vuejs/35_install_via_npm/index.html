Installation des modules via NPM

on initialise un projet nodejs avec la commande npm init.
un fichier de configuration package.json est généré.
Ce fichier est le patron, le plan, de chaque projet. Il permet de stocker des modules et leurs versions, des scripts, etc...

un fois le projet initialisé, on peut installer des modules avec la commande npm install nom-du-module.
Par exemple : (sudo) npm install vue.

Ce fichier json est exportable, il permet de distribuer votre app facilement, par exemple via github.
