JS - Révisions I

Nous avons vu déjà un certain nombre de concepts.
Ces bases sont importantes, nous continuerons à construire en s'appuyant sur les outils déjà abordés.

Pour faciliter la programmation de nos apps, installer un plugin sur votre éditeur de texte :
JSLint OU JSHints. Tester les deux et choisissez.

Concepts clés à comprendre et retenir :

  • La console pour déboguer
  • La documentation est en ligne
  • Les types de valeurs
  • Les tables de vérité (booléens)
  • Les opérateurs
  • Les variables
  • La syntaxe d'une fonction
  • Le(s) paramètre(s) d'une fonction
  • La valeur de retour d'une fonction
  • Le scope des fonctions
  • La portée des variables
